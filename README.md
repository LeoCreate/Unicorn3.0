# Unicorn 3.0 独角兽模板风格套系

#### 介绍

Unicorn3 独角兽模板风格套系 是 [魔法世界演绎论坛](http://www.hp-forum.com) 旗下 Unicorn Team 团队为 [魔法世界演绎论坛](http://www.hp-forum.com) 量身定制的 Discuz！X3.4 模板程序。

#### 安装教程

1. 下载模板文件并解压
2. 将模板文件夹 unicorn_3 上传至 Discuz 目录下的 template 文件夹内
3. 后台安装模板并启用
4. 导入模板的 DIY 文件
